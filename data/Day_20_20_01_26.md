### __Main types of graphs__

* Barplot 
Shows the average value per category.Compare numeric across categories
By default, it calculates the mean.
sns.barplot(data=df, x="city", y="income", hue="gender")

* Boxplot
Shows statistikal information (median,quartiles,outliers)
sns.boxplot(data=df, x="city", y="income" ,hue="gender" )

* Histogram
A histogram shows how numeric values spread..
sns.histplot(data=df, x="salary")

* Catplot
Compare multiple categories visually.
It’s used when:
-one variable is categorical
-the other is numeric
-you want to compare groups
don't work with plt.show, plt.savefig
sns.catplot(data=df, x="category", y="value", kind="box")

* Scatterplot
Shows individual data points.
Often used to explore correlations/relationships.
sns.scatterplot(data=df, x="age", y="salary")

* Lineplot
Mostly used for time series. 
sns.lineplot(data=df, x="date", y="sales")

* Relplot
It’s used when you want to analyze relationships between numeric variables.
Like scatterplot, but facets for multiple groups
kind can be scatterplot or line plot
Similarly to matplotlib's subplots, you can use relplot to create multiple plots next and on top of each other (parameters col="<split variable for columns>" and row="<split variable for rows>")
Relpot needs it's own parameters to determine figure size by using height= , aspect=
sns.relplot(data=df,x="x",y="y",kind="scatter", hue ="gender")


### __Main parameters of graphs__

* data
Data source.Our data frame/tables
data= name of data frame

* x, y
Column names used for the axes.
x="age"
y="salary"

* hue
Splits the data by color.
Each category gets a different color.
hue="gender"

* size
Controls the size of points.
Must be a numeric variable.
size="population"


### __Basic chart design tools__

The following tools are used to design the chart matplotlib

* Figure size
  plt.figure(figsize=(10, 5))

* Chart title
  plt.title("Sales by year") 

* Axis names
  plt.xlabel("Year")
  plt.ylabel("Sales")

* Axis limits
  plt.xlim(0, 100)
  plt.ylim(0, 5000)

* Ticks
  plt.xticks(np.arange(0,85,5))
  plt.yticks(np.arange(0,50,5)) for arranging the intrevals on our axes

* Rotate axis labels
  plt.xticks(rotation=45)

* Grid
  plt.grid(True)

* Legend
  plt.legend()

* Fonts
  plt.title("Title", fontsize=14)
  plt.xlabel("X", fontsize=12)

* Indents
  plt.tight_layout()

* Saving the graphs
  plt.savefig("plot.png", dpi=300, bbox_inches="tight")
