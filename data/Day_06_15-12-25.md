## Git Commands we learnt

* git clone <ssh-repo-link>
This is to clone a remote repo for the first time into your local PC

* git branch
This is to list down all the local branches that we have

* git status
This is to get the status of the current branch changes

* git add .
This is to add all the current working changes to the staging area

* git commit -m "write your message in here"
This is to add the commit message

* git push -u origin
This is to push our committed changes to the remote repo saved under the name "origin"

* git remote
To list down all the connected remote URLs

* git pull
to download all the changes from remote to local

* git checkout <branch_name>
To jump on the branch

* git switch <branch_name>
To jump on the branch

* git switch -c <branch_name>
To create the branch and jump onto it directly, one step instead of two commands

* git init
To initialise an empty repo. if you don't want to work off a cloud/remote one.

## Terminal commands

* cd <the name of the directory>
To jump into a directory

* cd ..
To jump up/back one directory / to get into the parent directory of our current folder

* ls
To list down all files and folders in the current directory

* mkdir <name of the desired folder>
To create a new folder

* pwd
To get the full path of your current directory
